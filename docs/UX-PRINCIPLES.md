# UX-PRINCIPLES.md

## Princípios

-   persona-aware by design
-   decision-oriented
-   governance-first
-   consistency over novelty
-   reduce cognitive load
-   contextual actions first
-   evidence before intuition

## Regras

-   Toda tela deve deixar claro o que o utilizador pode fazer.
-   O produto deve priorizar contexto operacional e governança.
-   A navegação deve refletir responsabilidade e escopo do utilizador.
