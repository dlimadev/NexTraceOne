## 2. Objective
Implement the `ICostIntelligenceModule` cross-module interface defined in `NexTraceOne.OperationalIntelligence.Contracts/Cost/ServiceInterfaces/ICostIntelligenceModule.cs`. The implementation must read real data from `CostIntelligenceDbContext` (which has migrations including `20260325210613_InitialCreate.cs` and `20260327072730_P6_4_CostRecord_ReleaseId.cs`). This interface enables other modules — particularly Governance (FinOps reports), Catalog (service cost context), and AIKnowledge (cost-aware AI analysis) — to query cost data without violating bounded context boundaries.

## 3. Problem actual
The `ICostIntelligenceModule` interface is defined with six methods and a `CostRecordSummary` DTO but has zero implementation:

```csharp
public interface ICostIntelligenceModule
{
    Task<decimal?> GetCurrentMonthlyCostAsync(string serviceName, string environment, CancellationToken ct = default);
    Task<decimal?> GetCostTrendPercentageAsync(string serviceName, string environment, CancellationToken ct = default);
    Task<IReadOnlyList<CostRecordSummary>> GetCostRecordsAsync(string? period = null, CancellationToken ct = default);
    Task<CostRecordSummary?> GetServiceCostAsync(string serviceId, string? period = null, CancellationToken ct = default);
    Task<IReadOnlyList<CostRecordSummary>> GetCostsByTeamAsync(string team, string? period = null, CancellationToken ct = default);
    Task<IReadOnlyList<CostRecordSummary>> GetCostsByDomainAsync(string domain, string? period = null, CancellationToken ct = default);
}
```

The `CostRecordSummary` DTO includes: ServiceId, ServiceName, Team, Domain, Environment, TotalCost, Currency, Period, Source.

`CostIntelligenceDbContext` has entities: `CostSnapshot`, `CostAttribution`, `CostTrend`, and `CostRecord` (added in P6_4 migration). Without this implementation, FinOps contextual views and cost-aware change intelligence are blocked.

## 4. Resultado esperado
- A `CostIntelligenceModule` class implementing `ICostIntelligenceModule` in the OperationalIntelligence Infrastructure project
- `GetCurrentMonthlyCostAsync` returns the latest monthly cost for a service in a specific environment
- `GetCostTrendPercentageAsync` returns the percentage change in cost compared to the previous period
- `GetCostRecordsAsync` returns all cost records for a period, aggregated into `CostRecordSummary`
- `GetServiceCostAsync` returns cost summary for a specific service
- `GetCostsByTeamAsync` returns cost summaries grouped by team
- `GetCostsByDomainAsync` returns cost summaries grouped by domain
- All queries are read-only with `AsNoTracking()`
- Registration in DI via the CostIntelligence module's DependencyInjection
